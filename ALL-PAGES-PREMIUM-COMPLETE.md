## 🎨 CONSISTENT DESIGN SYSTEM

### **Typography Hierarchy:**
```
Hero Headlines: 8xl-9xl font-black (72-96px)
Section Headers: 6xl-7xl font-black (48-60px)
Subheadings: 2xl-3xl font-light/semibold
Body: xl-2xl font-light
CTAs: xl font-black uppercase
```

### **Gradient Backgrounds:**
```css
Heroes: from-black to-bg-secondary
Content: from-bg-secondary via-black to-bg-tertiary
Flowing: Smooth transitions, no harsh breaks
```

### **Animated Elements:**
```
Gradient text: 4s infinite shimmer
Scale hover: 105% (350ms)
Active press: 95% (350ms)
Glow effects: 20-60px blur radius
```

### **Color Usage:**
```
Accent (#39FF14): CTAs, glows, key terms
White (#fafafa): Main text, emphasized words
Gray (#a0a0a0): Secondary text
Tertiary (#6b6b6b): De-emphasized text
Strikethrough: line-through + text-tertiary
```

### **CTA Styling:**
```css
Size: px-16 py-6
Font: text-xl font-black
Style: UPPERCASE tracking-wider
Effects: Dual gradients, scale, glow, shadow
Consistency: SAME across ALL pages
```
